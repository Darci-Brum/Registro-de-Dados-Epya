Preparado para Supabase

O arquivo supabase_schema.sql acompanha este ZIP com as tabelas sugeridas.

Próxima etapa para virar multiusuário real

Crie um projeto no Supabase, rode o SQL do pacote, configure autenticação e troque o armazenamento local por chamadas ao banco. O painel já foi separado por módulos para facilitar essa migração.

  • RDOs: tabela rdos
  • Despesas: tabela expenses
  • Equipes: tabela team_members
  • Veículo: tabelas vehicles e vehicle_costs
  • Agenda: tabela agenda_items
  • Usuários/perfis: tabela profiles
